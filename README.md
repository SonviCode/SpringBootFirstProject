# My first project with Java Spring

## A simple full stack application with the basic features of a db

## Stack :
- React.Js
- Tailwind CSS
- Typescript
- MySQL Workbeanch
- Spring Boot
- PostMan

### For start the project
Go to MySQL :
- create database with the same name
- change the password with your password

Go to backend : 
- run the application

Go to frontend and :
- Npm install
- Npm run dev
